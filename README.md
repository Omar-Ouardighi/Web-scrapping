The projects are

## Scraping [US Mayors](https://github.com/Omar-Ouardighi/Web-scrapping/tree/main/Us%20Mayors)

## Scraping [IMDB](https://github.com/Omar-Ouardighi/Web-scrapping/tree/main/Scraping%20IMDB)
